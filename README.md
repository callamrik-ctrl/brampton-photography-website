# Brampton GTA Photography Website

Static SEO-friendly photography website for Brampton and the Greater Toronto Area.

## Pages

- `index.html` - homepage
- `weddings.html` - wedding photography service page
- `packages.html` - wedding package and pricing page
- `engagement-family.html` - engagement, family, and portrait services
- `portfolio.html` - image portfolio
- `about.html` - photographer and service area page
- `contact.html` - inquiry form and FAQ

## Before Publishing

Replace these placeholders:

- `https://example.com` in canonical links, `robots.txt`, `sitemap.xml`, and homepage schema
- `Brampton GTA Photography` with the final studio name
- `416-555-0125` with the real phone number
- `hello@example.com` with the real email address
- Package prices if your actual pricing is different

The site can be opened directly from `index.html` or hosted on any static web host.
